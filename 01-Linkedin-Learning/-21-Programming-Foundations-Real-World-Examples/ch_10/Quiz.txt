1- Instead of adding or removing items from a list with a for loop, what should you do?

- Answer: Add or remove items from a copy of the list.

2- What the key difference between For and While loop?

- Answer: A For loop runs a set number of times, whereas a While loop runs a variable number of times.

3- What happens when a break statement is encountered in a For loop?

- Answer: The For loop is exited, then the program moves on to the next section of code.
