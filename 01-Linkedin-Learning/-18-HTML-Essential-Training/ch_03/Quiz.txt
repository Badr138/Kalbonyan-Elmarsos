2- Which of these attributes is NOT a global attribute?

- Answer: datetime
Correct. The datetime attribute is only used on the time element, as a way to create a machine-readable date.

3- When should we use an ARIA attribute?

- Answer: any time the existing markup makes things confusing in the accessibility tree
Exactly. ARIA is used to clarify to the accessibility tree what is happening with a particular element, set of elements, or interface. If something is broken, ARIA can be a way to fix it.

4- What sort of efforts require the use and understanding of ARIA attributes?

- Answer: those involving accessibility issues
